# Editor Musical para Microcontroladores

Editor de partituras web que genera código C/C++ para ESP32 y ATmega328P.

## Uso

Abre directamente en el navegador o visita la página web del proyecto.

## Microcontroladores soportados

- ESP32 (Arduino framework)
- ATmega328P (C puro, avr-gcc)

## Funciones

- Pentagrama interactivo con soporte de compases 2/4, 3/4, 4/4, 6/8
- Generación de código en tiempo real
- Exportar .ino / .c
- Exportar MIDI
- Guardado automático en localStorage
